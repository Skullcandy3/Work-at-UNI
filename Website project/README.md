## Wordle Web Application
This is a web-based implementation of the popular game Wordle, developed using Python Flask. Users can guess five-letter words within six attempts, and their progress is tracked through a login system. Points are awarded based on how quickly the word is guessed. The application manages users and word data through two SQLite databases. We used ( https://www.nytimes.com/games/wordle/index.html ) as inspiration for our own website, to help us create a fully working JavaScript game.


# Features
- Interactive Wordle gameplay
- User registration and login system
- Points system based on guessing performance
- Leaderboard to track top players
- Admin page for managing users 
- Persistent data management via SQLite
- Flask-based backend for server routing
- Dynamic CSS styling on webpage

# Prerequisites
- Python 3.x installed
- Flask installed (pip install flask)
- Flask-login installed (pip install flask-login)
- Werkzeug installed (pip install werkzeug)

# Setup Instructions
1. Clone the Repository
2. Open your code editor of choice
3. Make sure both users.db and words.db are in the root directory.
4. Run app.py code and make sure the terminal displays something
5. Start you browser engine of choice and insert: http://localhost:5000
